## 🤝 如何贡献

### 成为贡献者
如果您想为 MarkLens 项目做出贡献，可以：

1. **代码贡献**
   - Fork 项目并提交 Pull Request
   - 报告 Bug 和提出功能建议
   - 改进代码质量和性能

2. **文档贡献**
   - 完善项目文档
   - 编写使用教程
   - 翻译多语言文档

3. **社区贡献**
   - 帮助其他用户解决问题
   - 分享使用经验和技巧
   - 推广项目给更多人

4. **设计贡献**
   - 提供 UI/UX 设计建议
   - 创建图标和图形资源
   - 优化用户体验
